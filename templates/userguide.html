User Guide

After logging in, simply check the boxes for all the times when you are available. Remember, only leave boxes unchecked for when you are absolutely not free!

After pressing "Submit", you can view your updated availability under "Your Availability".

Admin Guide

After logging in, you will see the Admin dashboard. In order to allows users to add their availability, you must first:

  1. Add users: add the firstname, lastname, and email address and check the boxes if the user is a choregrapher or non-Harvard member
  2. Add pieces: define the cast list by checking the box of each dancer in the piece
  3. Define times: decide what days you want tech week to be
Next, you can run the scheduler. To do so, you will have to decide specifically when tech rehearsals should be and confirm all information.
